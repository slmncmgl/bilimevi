# Bilimevi Sözleşme Test Sistemi

Dijital sözleşme kabul platformu - Scroll tracking demo

## Özellikler

- PDF/DOCX yükleme
- Scroll tracking (%100 okunma zorunluluğu)
- Zaman takibi
- İsim doğrulaması
- Google Sheets entegrasyonu (yakında)

## Test Etmek İçin

1. Ana sayfada PDF yükleyin
2. "Sözleşme Oluştur" deyin
3. Açılan sayfada scroll yapın
4. %95'e gelince "Kabul Et" butonu aktif olur

## Stack

- Next.js 15
- TypeScript
- Vercel

## Geliştirme
```bash
npm install
npm run dev
```

Tarayıcıda: http://localhost:3000
